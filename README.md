# Hotel Reservation System

## Project Type
Console-based application developed as a self-learned project to explore hotel booking management.

## Overview
The **Hotel Reservation System** is a console-based application designed to manage hotel reservations and bookings. This project provided a hands-on experience in building a booking system, handling data, and creating a structured console application.

## Features
- **Room Availability Check**: View available rooms based on room type.
- **Room Booking**: Book a room by entering customer details.
- **Booking Management**: View, update, and cancel existing reservations.
- **Billing**: Generate bills based on room type and duration of stay.
